Notification System for any equipment in school.

This system consists of 3 main pages
  -First, Notify.html
  -Second, displayInfo.php
  -Third, behindSearch.php

For notify.html is the page that allows users to notify any types of damage which occur in their organization.
Next is displayInfo, this page will display all of notifications that sent to the system.
Final is behindSearch, will receive some keyword to classify the information that user want to search.
